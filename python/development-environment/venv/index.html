venv

virtual environment in development of python

Creating virtual environments

The following commands creates a folder for building a virtual environment in the .venv folder. 1

python3 -m venv .venv

Activate/deactivate virtual environment

Commands Details
source .venv/bin/activate activates virtual environment2
deactivate deactivate virtual environment

Install packages

Packages can be installed by using pip.
